# 자동차 경주 미션

## 기능 요구사항
- n대의 자동차가 참여할 수 있다.
- 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
  - 0에서 9 사이에서 random 값을 구한 후 random 값이 4 이상일 경우 전진하고, 3 이하의 값이면 멈춘다.
- 자동차 경주 게임을 완료한 후 누가 우승했는지를 구할 수 있다. 우승자는 한 명 이상일 수 있다.

## 구현할 기능 목록

### 도메인 모델
- [x] Car 클래스
  - 자동차 상태(이름, 현재 위치) 관리 및 이동
- [x] Race 클래스
  - 한 라운드 진행 로직
- [x] CarList 클래스
  - 자동차 리스트 관리
  - 우승자 선별

### 유틸리티 클래스
- [x] 랜덤 숫자 생성기
  - 0~9 사이의 난수 생성

### 입출력 계층
- [x] InputView 클래스
  - 자동차 개수 입력
  - 자동차 이름 입력
  - 라운드 횟수 입력
- [x] OutputView 클래스
  - 우승자 출력

### 컨트롤러 & 애플리케이션
- [x] Controller 클래스
  - 게임 흐름 관리
    - 자동차, 라운드 횟수 입력
    - 경주 실행
    - 우승자 판별
    - 우승자 출력
- [x] Application 클래스
  - 프로그램 진입점

## 유비쿼터스 언어
|용어| 설명 및 역할                                        |
|:--|:-----------------------------------------------|
|Car| 경주에 참여하는 자동차. 이름과 현재 위치 상태를 가진다.               |
|name| 자동차 이름                                         |
|position| 자동차의 현재 위치                                     |
|randomNumber| 전진 여부를 결정하기 위해 생성하는 0~9 사이의 정수                 |
|move| randomNumber가 기준값(4) 이상일 때, position을 1 증가시킨다. |
|carList|자동차를 모아놓은 리스트|
|playRoundAllCar|자동차 리스트 전체를 모든 라운드에 대해 실행시킨다.|
|findWinner|position이 가장 큰 Car 객체를 찾는다.|
|MIN_NUMBER| randomNumber가 생성될 수 있는 범위의 최솟값                 |
|MAX_NUMBER| randomNumber가 생성될 수 있는 범위의 최댓값                 |
|MOVE_THRESHOLD|자동차가 움직이기 위한 randomNumber 최소치|
